## Timer Actions Explained

### Action Types
1. **Turn ON** - Turns the light on at scheduled time
2. **Turn OFF** - Turns the light off at scheduled time
3. **Set Brightness** - Sets PWM light to specific brightness (shows extra field)

### Repeat Options
1. **Once** - Executes once at specified time, then deletes itself
2. **Daily** - Repeats every day at the same time
3. **Weekdays** - Repeats Monday through Friday
4. **Weekends** - Repeats Saturday and Sunday

## Timer States

### Active Timer
- Green left border
- Full opacity
- Shows â¸ (pause) button
- Will execute at scheduled time

### Inactive Timer
- Gray left border
- Reduced opacity (60%)

## Color Coding

- **Green border**: Active timer, will execute
- **Gray border**: Inactive timer, paused
- **Green button**: Timer is active, click to pause
- **Gray button**: Timer is inactive, click to resume
- **Red button**: Delete timer

## Time Display Format

- **One-time timers**: Shows date and time (e.g., "Feb 4 23:30")
- **Repeating timers**: Shows repeat pattern and time (e.g., "Daily at 18:30")
- **All times**: Displayed in 24-hour format

## Tips for Using Timers

1. **Test First**: Create a one-time timer a few minutes in the future to test
2. **Check Status**: System status shows count of active timers
3. **Pause, Don't Delete**: If you want to temporarily disable a timer, pause it instead of deleting
4. **Brightness Control**: Only works with PWM-enabled lights (Living Room, Kitchen, Bedroom, Bathroom by default)
5. **Time Format**: Use 24-hour format (e.g., 18:30 for 6:30 PM)
6. **Auto Refresh**: Timer list updates every 30 seconds automatically

## Common Use Cases

### Morning Routine
```
Kitchen: ON at 06:00 (Daily)
Bathroom: 50% at 06:00 (Weekdays)
```

### Evening Routine
```
Living Room: ON at 18:00 (Daily)
All Lights: OFF at 23:00 (Daily)
```

### Security/Vacation Mode
```
Living Room: ON at 19:00 (Daily)
Living Room: OFF at 23:00 (Daily)
Bedroom: ON at 21:00 (Daily)
Bedroom: OFF at 22:30 (Daily)
```

### Energy Saving
```
All Lights: OFF at 23:00 (Daily)
Outdoor Lights: ON at 18:00 (Daily)
Outdoor Lights: OFF at 06:00 (Daily)
```
